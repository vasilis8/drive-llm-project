Problem Statement

In high-performance autonomous racing, agents typically optimise a static objective (e.g., minimum lap time).

However, dynamic race scenarios demand adaptive strategies triggered by high-level instructions:

  • "Defend the inside line"
  • "Push for overtake"
  • "Conserve tires"

Goal

Develop an agent that maps natural language commands + raw LiDAR scans ‚Üí continuous steering & acceleration inputs.
